This is a budgeting tool that I created. It has an overview tab, a tab for budgets, and a tab to track subscriptions. A user can edit and delete budgets and subscriptions. It will also show if a user is going over the set amount for each budget, or if they are about to.

Tech:

  • ANGULARJS
  • SASS
  • TAILWIND
  • JASMINE
