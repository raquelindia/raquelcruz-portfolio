Deployed App

This is a project I worked on with a group. It is a CRUD app. A user can create, view, edit or delete each item in the inventory.

Tech:

  • REACT
  • SEQUELIZE
  • EXPRESS
  • RENDER

Objective

This is a group project for Multiverse. The objective was to create an e-commerce store with items that can be viewed, created, updated, or deleted.

Criteria:

  1. MVP Application: A user should be able to view all items in the inventory and any individual item.
  2. Adding an Item: A user should be able to add an item by completing a form.
  3. Deleting an Item: A user should be able to remove an item from the inventory
  4. Updating an Item: A user should be able to update an item by filling a form

Approach

The group I was working on this project with decided that they wanted to use React. I did the wireframes and the CSS.

Steps Taken

The first thing we did was choose a color scheme. I found one on google that everyone liked, so we went with that one and then I created the wireframes. Most of the work done in this project was done by another apprentice. We did this project at the the beginning of the apprenticeship and we had not done anything with React, which is what we chose to use for this project, so I did most of the CSS. I did have to get familiar with React because the way we were using CSS with it was different than what I was used to seeing. This wasn't a very complicated project, We used stock images for the items and put a form on each individual page to edit or delete an item and we used Render to deploy it.
