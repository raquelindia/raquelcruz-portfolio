Objective

This is a group project for Multiverse. The objective was to create an e-commerce store with items that can be viewed, created, updated, or deleted.

Criteria:

  1. MVP Application: A user should be able to view all items in the inventory and any individual item.
  2. Adding an Item: A user should be able to add an item by completing a form.
  3. Deleting an Item: A user should be able to remove an item from the inventory
  4. Updating an Item: A user should be able to update an item by filling a form

Approach

We built this project using React, Sequelize, Express, and deployed it using Render. We split the work up really based on our strengths.

Steps Taken

This project took us 4 weeks to finish. The first thing we did was choose a color scheme. I found one on google that everyone liked, so we went with that one and then I created the wireframes. This wasn't a very complicated project, We used stock images for the items and put a form on each individual page to edit or delete an item and then we used Render to deploy it.

Challenges

The only challenge for me during this project was working with React. We did this project at the the beginning of the apprenticeship and we had not done anything with React yet, so I did most of the CSS. I did have to get familiar with React a little because the way we were using CSS with it was different than what I was used to seeing.

Summary

Something I learned while working on this project was CSS works differently with different frameworks. I think this was the first group project we did during the Multiverse bootcamp and there were a lot of concepts that were still very new to me. Having to use "className" instead of classes was confusing at first especially since I had never used React prior to this. Overall, I learned a lot from working on this, mostly from the people I was working with.
