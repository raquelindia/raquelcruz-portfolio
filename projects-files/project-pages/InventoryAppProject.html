This is a project I worked on with a group. It is a CRUD app. A user can create, view, edit or delete each item in the inventory.

Tech:

  • REACT
  • SEQUELIZE
  • EXPRESS

Objective

This is a group project for Multiverse. The objective was to create an e-commerce store with items that can be viewed, created, updated, or deleted.

Criteria:

  1. MVP Application: A user should be able to view all items in the inventory and any individual item.
  2. Adding an Item: A user should be able to add an item by completing a form.
  3. Deleting an Item: A user should be able to remove an item from the inventory
  4. Updating an Item: A user should be able to update an item by filling a form
