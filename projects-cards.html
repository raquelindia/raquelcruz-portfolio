Journal API

This is an API I created. In this API you can log in using Auth0. You can CREATE, READ, UPDATE, and DELETE a journal entry. If you have a SUPERADMIN account, you can READ, UPDATE, and DELETE any entry from any user, and UPDATE, or DELETE any user account.
